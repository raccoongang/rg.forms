How Backend Validation Works


    
1. visible_when - Skipping Hidden Fields

    
priority = ReactiveChoiceField(
    visible_when="$order_type == 'urgent'",
    required=True,  # Required, but only validated when visible!
)

# In ReactiveForm._clean_fields():
if not self.is_field_visible(name):
    self.cleaned_data[name] = None  # Skip validation
    continue


    
2. required_when - Enforcing Dynamic Requirements

    
bulk_discount_code = ReactiveCharField(
    required=False,
    required_when="$order_type == 'bulk'",
)

# Backend evaluates expression and raises ValidationError if empty


    
3. computed - Server-side Recalculation

    
total = ReactiveDecimalField(
    computed="$quantity * $unit_price",
)

# Backend ignores submitted value, recalculates:
computed_value = self.get_computed_value(name)
if computed_value is not None:
    value = computed_value  # Override submitted value
